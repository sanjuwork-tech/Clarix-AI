## Quick Audit: What's Working vs What Needs Fixing

### Keep as-is
- Horizontal scroll direction — correct
- Stick figure visual language — on brand
- Clarix diagnosis scene amber glow — strong
- Green checkmark PASS scene with confetti — excellent
- The "repeat?" circular arrow — clever, keep it
- Overall parchment background (#FAF8F4) — correct

### Fix now (in priority order)
1. Remove ALL year/date references — critical for audience reach
2. Stick figures have zero emotion — all look identical
3. Background is empty between scenes — no environment, no depth
4. Stage labels need rewriting
5. Scene 1 starts at Coaching Institute — should start at BCom/Start
6. Color has no progression — should build from neutral → warm stress → amber Clarix → green PASS
7. The path/ground connector dots are too subtle
8. Stick figure walking animation missing between scenes
9. Fonts on scene labels too small and muted
10. The section header "THE CA ASPIRANT LOOP — ANIMATED" is too literal

## Part 11: Performance Notes

The animation has multiple looping Framer Motion elements. To prevent jank:

```tsx
// Add to every motion element that loops continuously:
style={{ willChange: 'transform' }}

// For the main horizontal scroll container — use CSS scroll behavior:
// Do NOT use Framer Motion for the horizontal scroll itself
// Use CSS scroll-snap for better performance:
style={{
  display: 'flex',
  overflowX: 'auto',
  scrollSnapType: 'x mandatory',
  scrollBehavior: 'smooth',
  WebkitOverflowScrolling: 'touch',  // iOS momentum scroll
}}

// Each scene section:
style={{
  scrollSnapAlign: 'start',
  flexShrink: 0,
}}

// Hide scrollbar but keep functionality:
// Add to your CSS file:
.scroll-container::-webkit-scrollbar { display: none; }
.scroll-container { -ms-overflow-style: none; scrollbar-width: none; }
```
